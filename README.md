# Image Compressor

A simple client-side image compression tool built with Next.js. Compress JPG and PNG images directly in your browser without uploading files to any server.

## Features

- Client-side image compression
- No server uploads
- Fast and simple
- Works on modern browsers

## How It Works

- Upload an image file
- Image is processed in the browser using the Canvas API
- Compressed image is generated locally
- Download the result

## Tech Stack

- Next.js
- JavaScript
- HTML Canvas API

## Local Development

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Privacy Note

All image processing happens locally in your browser. No images are uploaded to any server or stored anywhere. Your images never leave your device.
